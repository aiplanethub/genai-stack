# LLM Stack

An end to end LLM framework

-   Free software: BSD license
-   Documentation: https://llm_stack.readthedocs.io.

## Usage

1. Install with pip
    ```bash
    pip install git+https://github.com/aiplanethub/llm-stack.git
    ```
2. Verify installation
    ```bash
    llmstack version
    ```
3. Check List of commands
    ```bash
    llmstack --help
    ```
