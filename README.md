# LLAIM

An end to end LLM framework

-   Free software: BSD license
-   Documentation: https://llaim.readthedocs.io.

## Usage

1. Install with pip
    ```python
    pip install git+https://github.com/dphi-official/llaim.git
    ```
2. Check List of commands
    ```python
    llaim --help
    ```
